================================================================================
  TROUBLESHOOTING
================================================================================

Issue: "Flutter not found"
Solution: Add Flutter to PATH or reinstall

Issue: "Pod install fails" (iOS)
Solution: cd ios && rm -rf Pods && pod install --repo-update && cd ..

Issue: "Gradle build fails" (Android)
Solution: flutter clean && flutter pub get && flutter build apk --release

Issue: "Socket.IO connection fails"
Solution: Update API_URL in lib/config/app_config.dart

Issue: "Version code must be higher"
Solution: Increment build number in pubspec.yaml

For more troubleshooting, see BUILD_GUIDE.md
